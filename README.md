# portfolio

Personal portfolio site at [esawadhana.com](https://esawadhana.com).

## Overview

Single-page Next.js portfolio with sections rendered as App Router routes. Includes a contact form wired to EmailJS and motion-driven UI via Framer Motion. Hosted on Vercel; custom domain configured via `CNAME`.

## Stack

- Next.js + React + TypeScript
- Tailwind CSS
- Framer Motion
- `@emailjs/browser` (contact form)
- `react-hook-form` + Lucide icons

## Getting started

```bash
npm install
npm run dev
```
